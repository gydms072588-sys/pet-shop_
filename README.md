# PAVONIQ 애견 가구/소품 쇼핑몰

PAVONIQ는 애견 가구/소품 쇼핑몰 웹사이트를 Cafe24 쇼핑몰 스킨 적용 가능성을 고려해 정적 HTML/CSS/JS로 구성한 프로젝트입니다.

## 주요 페이지
- `index.html`: 메인
- `products.html`: 상품 목록
- `product-detail.html`: 상품 상세
- `cart.html`: 장바구니
- `wishlist.html`: 관심상품
- `login.html`: 로그인
- `signup.html`: 회원가입

## 주요 폴더
- `css/`: 공통 스타일
- `js/`: 공통 인터랙션
- `assets/images/`: 상품, 배너, 상세페이지 이미지
