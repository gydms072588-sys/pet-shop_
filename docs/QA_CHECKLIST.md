# QA Checklist _QA 체크리스트

이 문서는 PAVONIQ 애견 가구/소품 쇼핑몰 정적 HTML 작업물을 검수하기 위한 체크리스트입니다. Cafe24 쇼핑몰 스킨 적용 전, 현재 파일 상태 기준으로 화면 구조와 링크, 이미지, 반응형 상태를 확인합니다.

## 1. 공통 QA
- [ ] 모든 페이지에서 상단 프로모션 바가 동일하게 노출되는지 확인
- [ ] 모든 페이지에서 공통 헤더가 노출되는지 확인
- [ ] 모든 페이지에서 공통 푸터가 노출되는지 확인
- [ ] 헤더 로고가 `PAVONIQ`로 표기되는지 확인
- [ ] 푸터 로고, 회사명, copyright, 고객센터 이메일의 브랜드 표기가 `PAVONIQ` 기준으로 일관적인지 확인
- [ ] 로고 클릭 시 `index.html`로 이동하는지 확인
- [ ] 주요 네비게이션 링크가 의도한 페이지 또는 앵커로 이동하는지 확인
- [ ] 헤더 아이콘 링크가 각각 검색, 로그인, 위시리스트, 장바구니 의미로 노출되는지 확인
- [ ] 버튼과 링크 hover 상태가 시각적으로 확인되는지 확인
- [ ] 상품 카드의 하트 버튼 클릭 시 상태가 변경되는지 확인
- [ ] `assets/images/` 하위 이미지 경로가 깨지지 않는지 확인
- [ ] Google Material Symbols 아이콘이 정상적으로 렌더링되는지 확인
- [ ] 폰트, 컬러, 여백, 버튼 스타일이 페이지 간 일관적인지 확인
- [ ] 페이지 하단 `맨 위로` 버튼이 노출되고 클릭 시 상단으로 이동하는지 확인
- [ ] 모바일 메뉴 버튼 클릭 시 메뉴가 열리고 닫히는지 확인

## 2. 메인 페이지 QA (`index.html`)
- [ ] 히어로 영역의 배경 이미지와 메인 카피가 정상 노출되는지 확인
- [ ] 히어로 CTA `컬렉션 보기` 클릭 시 `products.html`로 이동하는지 확인
- [ ] 서비스 스트립의 무료배송, 쉬운 반품, 안전한 결제, 고객센터 항목이 정렬되어 보이는지 확인
- [ ] 펫 가구 카테고리 섹션의 6개 카테고리 카드가 정상 노출되는지 확인
- [ ] 베스트 셀러 섹션의 상품 카드 이미지, 상품명, 가격이 읽기 쉽게 보이는지 확인
- [ ] 프로모션 배너 4장이 정상 노출되고 슬라이드/도트 상태가 동작하는지 확인
- [ ] 프로모션 배너 이미지 비율이 PC와 모바일에서 깨지지 않는지 확인
- [ ] 펫 라이프 가이드, 신상품 섹션의 상품 카드 링크가 `product-detail.html`로 연결되는지 확인
- [ ] 멤버십 배너 CTA가 `signup.html`로 이동하는지 확인

## 3. 상품 목록 페이지 QA (`products.html`)
- [ ] 페이지 히어로 영역에 `가구` 제목과 설명이 노출되는지 확인
- [ ] PC 화면에서 필터 영역과 상품 목록이 좌우 레이아웃으로 정렬되는지 확인
- [ ] 필터 영역의 카테고리, 반려동물, 사이즈, 가격대 UI가 깨지지 않는지 확인
- [ ] 상품 카드가 그리드로 일정하게 정렬되는지 확인
- [ ] 상품 이미지가 정사각형 비율을 유지하는지 확인
- [ ] 상품 브랜드명, 상품명, 가격 영역의 가독성이 유지되는지 확인
- [ ] 정렬 셀렉트 박스가 노출되는지 확인
- [ ] 페이지네이션 UI가 하단에 정상 노출되는지 확인
- [ ] 모바일 화면에서 필터가 숨겨지고 상품 카드가 1열 또는 2열로 자연스럽게 정리되는지 확인

## 4. 상품 상세 페이지 QA (`product-detail.html`)
- [ ] 대표 이미지와 `BEST` 배지가 정상 노출되는지 확인
- [ ] 상품명 `모던 우드 펫 소파 베드`, 브랜드, 설명, 평점, 가격 정보가 정상 노출되는지 확인
- [ ] 무료배송, 회원 혜택, 안심 서비스 안내가 깨지지 않는지 확인
- [ ] 컬러 옵션과 사이즈 옵션 버튼이 정상 배치되는지 확인
- [ ] 수량 조절 버튼 클릭 시 수량이 1 이상으로 변경되는지 확인
- [ ] `장바구니 담기`, `바로 구매하기`, 위시리스트 버튼 UI가 정상 노출되는지 확인
- [ ] 상세 탭 4개가 순서대로 노출되는지 확인: 상품 상세정보, 상품 리뷰, 상품 문의, 배송/교환/반품
- [ ] 탭 클릭 시 활성 탭과 패널이 정상 변경되는지 확인
- [ ] 상품 상세정보 탭에 `detail-page-01.png`부터 `detail-page-13.png`까지 순서대로 노출되는지 확인
- [ ] 상세페이지 13번 이미지가 `assets/images/detail-page/detail-page-13.png`로 교체된 최신 이미지인지 확인
- [ ] 상세 이미지들이 중앙 정렬되고 원본 비율로 표시되는지 확인
- [ ] 모바일에서 상품 이미지, 상품 정보, 상세 이미지가 세로 흐름으로 자연스럽게 배치되는지 확인

## 5. 장바구니 QA (`cart.html`)
- [ ] 장바구니 제목과 담긴 상품 개수가 노출되는지 확인
- [ ] 장바구니 상품 4개의 이미지, 상품명, 옵션, 가격이 정상 노출되는지 확인
- [ ] 전체선택, 선택삭제 UI가 노출되는지 확인
- [ ] 각 상품의 수량 조절 버튼 UI가 정상 동작하는지 확인
- [ ] 삭제 버튼 클릭 시 해당 행이 제거되고 상품 개수 표시가 변경되는지 확인
- [ ] 주문요약 카드의 상품금액, 배송비, 할인금액, 결제 예정 금액이 읽기 쉽게 보이는지 확인
- [ ] 주문하기 CTA와 네이버 페이 CTA가 정상 배치되는지 확인
- [ ] 함께 보면 좋은 추천 상품 섹션이 하단에 노출되는지 확인
- [ ] 현재는 정적 샘플 데이터이므로 실제 금액 재계산은 미구현 상태임을 확인
- [ ] 상품이 모두 삭제된 뒤 별도 빈 상태 화면은 아직 없는 상태임을 확인

## 6. 관심상품 QA (`wishlist.html`)
- [ ] 위시리스트 페이지 히어로와 설명 문구가 정상 노출되는지 확인
- [ ] 관심상품 카드의 이미지, 브랜드, 상품명, 가격, 평점이 정상 노출되는지 확인
- [ ] 하트 버튼이 선택된 상태로 노출되는지 확인
- [ ] 전체선택, 선택삭제 버튼 UI가 노출되는지 확인
- [ ] 위시리스트 요약 카드의 총 상품 수와 총 상품 금액이 읽기 쉽게 보이는지 확인
- [ ] 선택 상품 장바구니 담기, 전체 상품 장바구니 담기 CTA가 `cart.html`로 연결되는지 확인
- [ ] 재입고 알림 안내와 리스트 공유 문구가 깨지지 않는지 확인
- [ ] 현재는 정적 샘플 데이터이므로 실제 선택/삭제/공유 기능은 미구현 상태임을 확인
- [ ] 빈 위시리스트 상태 화면은 아직 없는 상태임을 확인

## 7. 로그인 QA (`login.html`)
- [ ] 로그인 폼이 화면 중앙에 안정적으로 배치되는지 확인
- [ ] 이메일 또는 아이디, 비밀번호 입력 필드가 정상 노출되는지 확인
- [ ] 비밀번호 필드의 visibility 아이콘이 깨지지 않는지 확인
- [ ] 로그인 상태 유지 체크박스가 정상 노출되는지 확인
- [ ] 로그인 버튼 위치와 크기가 모바일에서도 적절한지 확인
- [ ] 아이디 찾기, 비밀번호 찾기 링크가 노출되는지 확인
- [ ] 카카오, 네이버, 애플, 구글 간편 로그인 아이콘이 정상 노출되는지 확인
- [ ] 회원가입 패널의 브랜드 문구가 `PAVONIQ`로 표기되는지 확인
- [ ] 현재는 정적 폼이므로 실제 인증 기능은 미구현 상태임을 확인

## 8. 회원가입 QA (`signup.html`)
- [ ] 회원가입 제목과 소개 문구가 정상 노출되는지 확인
- [ ] 좌측 소개 이미지와 브랜드 카피가 PC에서 정상 노출되는지 확인
- [ ] 이름, 이메일, 비밀번호, 비밀번호 확인, 휴대폰 번호, 반려동물 종류 입력 영역이 정렬되어 보이는지 확인
- [ ] 약관 동의 체크박스들이 읽기 쉽게 노출되는지 확인
- [ ] 회원가입 버튼 위치와 크기가 적절한지 확인
- [ ] 이미 회원인 경우 로그인 링크가 `login.html`로 이동하는지 확인
- [ ] 회원 혜택 카드가 PC와 모바일에서 깨지지 않는지 확인
- [ ] 모바일 화면에서 좌측 소개 이미지는 숨겨지고 폼 중심으로 정리되는지 확인
- [ ] 현재는 정적 폼이므로 실제 회원 생성 기능은 미구현 상태임을 확인

## 9. 반응형 QA
- [ ] PC 기준으로 `container` 최대 폭과 섹션 간격이 안정적으로 보이는지 확인
- [ ] 1180px 이하에서 모바일 메뉴 구조가 정상 동작하는지 확인
- [ ] 1100px 이하에서 상품 그리드, 푸터, 상세 레이아웃이 무리 없이 줄어드는지 확인
- [ ] 820px 이하에서 주요 페이지 레이아웃이 1열 구조로 정리되는지 확인
- [ ] 520px 이하에서 버튼 텍스트, 탭, 상품 카드가 부모 영역을 넘치지 않는지 확인
- [ ] 모바일에서 가로 스크롤이 발생하지 않는지 확인
- [ ] 모든 상품 이미지와 상세 이미지가 찌그러지지 않고 비율을 유지하는지 확인
- [ ] 터치 가능한 버튼과 아이콘의 크기가 충분한지 확인
- [ ] 상세 탭은 모바일에서 가로 스크롤 가능한 구조로 유지되는지 확인

## 10. 운영 관점 QA
- [ ] Cafe24 적용 시 공통 헤더, 푸터, 상단 프로모션 바를 분리하기 쉬운 구조인지 확인
- [ ] 상품 카드, 상품 상세, 장바구니 행을 Cafe24 반복 영역으로 치환할 수 있는지 확인
- [ ] 이미지 파일명이 운영자가 이해할 수 있는 규칙으로 관리되는지 확인
- [ ] 상세페이지 이미지는 `assets/images/detail-page/`에 모여 있는지 확인
- [ ] 상품 이미지는 `assets/images/product-hq-*.jpg` 규칙으로 관리되는지 확인
- [ ] 배너 이미지는 `assets/images/main-banner-*.png` 규칙으로 관리되는지 확인
- [ ] 유지보수 시 수정 범위가 HTML, CSS, JS, 이미지 폴더 기준으로 명확한지 확인
- [ ] 실제 운영 전 이미지 용량 최적화가 필요한지 확인
- [ ] 실제 운영 전 개인정보, 사업자 정보, 고객센터 정보가 실제 정보로 교체되어야 하는지 확인
